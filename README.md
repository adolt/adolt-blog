# adolt-blog

## 介绍

跟着 [@nswbmw](https://github.com/nswbmw) 教程开发的个人博客

## 实现

项目中使用到：
- Node.js（运行环境）
- Express（后端框架）
- MongoDB（数据库）
- Ejs（服务器端渲染模板引擎）
- Sementic-UI（UI库）
- jQuery
